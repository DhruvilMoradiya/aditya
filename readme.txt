મોરડીયા સર ગુજરાતના એક માત્ર શિક્ષક કે જેમણે ફાઇલ પ્રોજેકટ દ્વારા કોમર્સના વિદ્યાર્થીઓને હિસાબી પ્રેકિટકલ શિક્ષણ આપ્યું છે. જેમણે ૨૫ વર્ષના શિક્ષણ કાર્યકાળમાં અસંખ્ય CA,C.s, હિસાબનીશો અને સફળ ઉદ્યોગવીરો સમાજને આપ્યા છે. હવેથી અનુભવી મોરડીયાસરના અભિનવ અનુભવનો લાભ અભિનવ સ્કૂલના વિદ્યાર્થીઓને મળશે.
